Historial de Sistemas Operativos

Década de 1950

• En esta época temprana de la computación, los sistemas operativos eran simples y estaban diseñados principalmente para administrar hardware especifico. Ejemplos: incluyen el Sistema Operativo de Control de Procesos (Batch) de IBM y el Sistema de Control de Cinta (Tape Control System) de UNIVAC.

Década de 1960

• IBM lanzo el Sistema Operativo/360 (OS/360) en 1964, uno de los primeros sistemas operativos que permitía la portabilidad de programas entre diferentes modelos de computadoras IBM.


• El sistema operativo Unix se origino en los laboratorios Bell de AT&T en la década del 1960.Fue muy importante para la historia del sistema operativo.

Década de 1980

• Microsoft lanzo Windows en 1985, una interfaz grafica que se ejecutaba sobre MS-DOS, sentado las bases para la popularidad de Windows en las PC.


• El sistema operativo Unix se diversifico en varias variantes, incluyendo BSD y System V.

Década de 1990

• Linux, un sistema operativo de código abierto basado en Unix, gano popularidad y se convirtió en una alternativa viable a los sistemas comerciales.


• Windows 95 y Windows NT de Microsoft introdujeron mejoras significativas en la interfaz gráfica y la estabilidad del sistema.

Década de 2000

• Mac OS X, basado en Unix, se convirtió en el sistema operativo principal de las computadoras Mac de Apple.


• Microsoft lanzo Windows XP, que se convirtió en uno de los sistemas operativos mas utilizados en todo el mundo.

Década de 2010 en adelante

• Windows 7,8,8.1y 10 sucedieron a Windows XP y se convirtieron en versiones populares de Microsoft.
